# Project Sprint 1 – Análisis de Datos

## 🧩 Problema
Realizar un análisis exploratorio de un conjunto de datos para identificar patrones, relaciones entre variables y posibles insights de negocio.

## 🧪 Mi rol
Como analista de datos:
- Realicé la exploración inicial del dataset.
- Identifiqué valores faltantes y estructuras de los datos.
- Generé visualizaciones para comprender distribuciones.
- Analicé relaciones entre variables clave.

## 🛠️ Herramientas utilizadas
- Python  
- Pandas  
- Matplotlib  
- Jupyter Notebook

## 📈 Resultados
- Se identificaron tendencias relevantes dentro del conjunto de datos.
- Se detectaron variables con mayor influencia en el comportamiento analizado.
- Se generaron conclusiones iniciales para futuras etapas de modelado.
